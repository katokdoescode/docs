Astro поддерживает переходы между страницами всего в несколько строк кода при помощи View Transitions. View transitions обновляют содержимое страницы при этом не обновляют всю страницу как при обычном переходе на новую страницу в бразуере, и обеспечивают плавную анимацию между страницами.
